# Resource Monitor  
Terminal-Based System Resource Monitor

---

## Overview
Resource Monitor is a lightweight, modular, terminal-based dashboard built using Bash.  
It provides real-time monitoring of:

- CPU usage  
- Memory usage  
- Disk I/O  
- Network throughput  
- Processes (with safe kill option)

The project is designed for learning, OS demonstrations, and simple system diagnostics.

---

## How to Run on Any Linux Machine

```bash
git clone https://github.com/isthisdeception/resource-monitor.git
cd resource-monitor
bash system_monitor.sh
